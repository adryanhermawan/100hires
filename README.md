# 100Hires Portfolio Project

This is my public portfolio project for the 100Hires Junior Growth Marketing Specialist recruitment process.

## Tools I Installed

* Cursor IDE
* Claude Code extension in Cursor
* Codex extension in Cursor
* GitHub

## Steps I Completed

First, I installed Cursor IDE from the official Cursor website. After installing it, I signed up to Cursor and chose to sign in using my Google account.

During the setup process, Cursor offered me a subscription plan, but I chose to skip it for now. So, at this stage, I am using the free plan. Cursor also asked me whether I wanted to connect GitHub at the beginning, and I chose “maybe later”.

After that, I tried to find where I could install the Claude Code and Codex extensions. I clicked the customize area and opened the plugin tab. I clicked “add” and started looking for the Claude Code and Codex extensions, but at first I could not seem to find them.

Because of that, I searched on Google for how to install Claude Code and Codex extensions in Cursor, step by step.

Later, I realized that I made a mistake in the previous step. The reason I could not find the extensions was because I was opening Cursor Agents instead of the main Cursor application.

After opening the correct Cursor application, I found that Claude Code could be installed through the extension panel. I opened the extension panel by pressing `Ctrl + Shift + X`, typed “Claude Code”, found the extension, and clicked install.

After installing Claude Code, instead of logging in immediately, I tried to find Codex first so I could install both extensions and log in to them later. I successfully installed Codex as well.

Next, I tried to log in to both Claude Code and Codex, but at first I could not find where to sign up or sign in. So I searched on Google again for how to sign in to Claude Code in Cursor.

I found several sources, but the one that worked for me was a YouTube video titled “How to Add Claude Code in Cursor (2026)”. From that video, I learned that to open Claude Code, I needed to click the three dots on “More Actions” in the top right area of Cursor.

After opening Claude Code, I clicked “Claude.ai Subscription”. This opened my browser. I chose to sign in with Google and selected the free plan.

Using the same method, I tried to log in to Codex. Since I already subscribe to ChatGPT Plus, I was able to sign in to Codex more easily.

The next step was creating a public GitHub repository. This step was easier for me because I already had a GitHub account that I created a long time ago, when I was learning about cloud technology.

After creating the public GitHub repository, I tried to open it in Cursor. Since I did not know how to do that at first, I searched on Google again.

I found that to open my public GitHub repository in Cursor, I needed to copy the repository URL from GitHub. Then, I opened Cursor on my computer, pressed `Ctrl + Shift + P` to open the Command Palette, typed `Git: Clone`, and selected it.

After that, I pasted my GitHub repository URL and pressed Enter. Then, I chose a destination folder on my local computer to save the files. Once the download finished, I clicked the “Open” prompt that appeared in the bottom right corner.

 I created this `README.md` file inside the repository to document what tools I installed, what steps I completed, and what issues I ran into during the setup process.

 Finally, I Commit this README.md file and push to GitHub

## What I Learned

From this process, I learned that setting up new AI tools can require some troubleshooting. Not every tool appears immediately in the place where I expect it to be. Sometimes I need to search, compare instructions, watch tutorials, and test the steps myself.

I also learned how to install and log in to Claude Code and Codex inside Cursor, how to create a public GitHub repository, and how to open that repository in Cursor by using `Git: Clone`.

This project is a simple first step, but it shows that I can follow technical instructions, solve setup issues independently, and clearly document my process.
